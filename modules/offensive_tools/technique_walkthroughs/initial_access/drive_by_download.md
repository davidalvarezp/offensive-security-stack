# Drive-By Download (Educational Walkthrough)

## Purpose
Explain the concept of unintended software execution via user interaction.

---

## Concept Overview
A drive-by download occurs when:
- A user visits a resource
- Content triggers an unintended action
- Execution happens without explicit consent

---

## Risk Factors
- Outdated software
- Unsafe browser configurations
- Untrusted content sources

---

## Defensive Controls
- Browser isolation
- Patch management
- Content filtering

---

## Teaching Objective
Understand **how user exposure leads to compromise**.
